# Localization and Mapping System for Autonomous Inspection Robots

AutoDRIVE is envisioned to be an integrated platform for autonomous driving research and education. 3D Mapping System is an open-source project providing 3D-printable modular parts, sensor initialization scripts, and SLAM system integration for creating versatile 3D mapping solutions for both robots and handheld devices. ConceptFusion is a framework designed for open-set multimodal 3D mapping, enabling the integration of diverse data modalities into a unified 3D representation. Nav2 is the go-to industry-standard mobile robot navigation system, deploying Autonomous Vehicle technologies brought down, reworked, and optimized for mobile and surface robotics. GLIM is a versatile and extensible range-based 3D mapping framework. OctoMap is an efficient probabilistic 3D mapping framework based on octrees. ROS Based 3D Mapping Rover with User Interface. Multi-Sensor 3D mapping with an unmanned ground vehicle for mining applications. Laser_RobMap is a ROS2 package designed for three-dimensional mapping using a two-dimensional LiDAR sensor mounted on a mobile robotic platform.
