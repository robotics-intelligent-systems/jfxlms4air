# Localization and Mapping System for Autonomous Inspection Robots

Nav2 is the go-to industry-standard mobile robot navigation system, deploying Autonomous Vehicle technologies brought down, reworked, and optimized for mobile and surface robotics. OctoMap is an efficient probabilistic 3D mapping framework based on octrees. ROS Based 3D Mapping Rover with User Interface. Multi-Sensor 3D mapping with an unmanned ground vehicle for mining applications. Laser_RobMap is a ROS2 package designed for three-dimensional mapping using a two-dimensional LiDAR sensor mounted on a mobile robotic platform.
